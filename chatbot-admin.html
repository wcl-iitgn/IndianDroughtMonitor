Administrator sign-in


This console talks to the WCL OpenAI API and only works for accounts the server recognises as administrators. Nothing is stored here except the session token (in this tab, cleared when it closes). This page is intentionally not linked from the site — but the real protection is the server-side admin check, not the hidden URL.

Add a user

The password is hashed by the server before it is stored. To change the seeded default admin: create a new admin account here, sign in with it, then delete the old admin row.
